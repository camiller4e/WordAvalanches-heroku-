Word Avalanches

About This Project

Hi! My name is Campbell and I'm a software developer based in Glasgow, Scotland. This site is the result of a Javascript project for CodeClan.

The brief was to create something (anything really, within reason) from an API, so I decided to create this convoluted puzzle, using the Word Avalanches subreddit. If you like the site, feel free to hit me up on GitHub.

If you've written any of the Word Avalanches that feature here and would like to be credited or if you'd like to have your Avalanche removed from the site, you can contact me via GitHub. Happy Avalanching!
