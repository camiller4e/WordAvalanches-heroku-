About This Project

Hi! My name is Campbell and I'm a software developer based in Glasgow, Scotland. This site is the result of a Javascript project for CodeClan.

The brief was to create something (anything really, within reason) from an API, so I decided to create an API using the Word Avalanches subreddit. If you like the site, feel free to hit me up on Twitter or follow me on GitHub.

If you've written any of the Word Avalanches that feature here and would like to be credited or have your Avalanche removed from the site, you can contact me using the links below. Happy Avalanching!
